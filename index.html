about paintbros

Welcome to paintbros, a retro bitmap graphics editor.

This is an early alpha version, so only a few basic features have been implemented so far, including drawing, erasing, and the color picker.

Check back for new features and improvements, and if you would like to contribute to this project, check out the github page at:

https://github.com/TheInnovationGroup/paintbros
